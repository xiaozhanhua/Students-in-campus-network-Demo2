# 移动端学生校园网（支持响应式）
>这是自己大三第一学期末的毕业设计项目，使用到的技术有：Bootstrap、jQuery、jsp、Struts2、Hibernate、Spring、MySql

###主要模块：
####论坛社区：提供看帖、发帖、回帖功能
####二手市场：提供产品发布、展示、搜索功能
####登录注册：实现用户帐号校验
